Personal Portfolio Website

This project is a personal portfolio website designed to showcase my skills, education, and the various software projects I have developed. It serves as a central hub for anyone who wants to learn more about my work as a developer.

Key Sections:

About Me: A brief introduction to my background as a Computer Science student at Jimma University.

Project Gallery: A display of my main projects including links to their live demos and source code.

Skill Set: A list of programming languages and tools I am proficient in, such as PHP, Kotlin, and MySQL.

Contact Form: A way for visitors and potential employers to reach out to me directly.

Technology Used:

Frontend: HTML5, CSS3, and JavaScript.

Design: Focused on a clean, responsive layout that works on both mobile and desktop.

Hosting: Deployed using GitHub Pages.

Live Preview: https://amiro-dev5.github.io/amiro-portfolio
